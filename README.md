# ready-4-ai

## Start zadania 2 (JavaScript + Node.js)

1. Zainstaluj Node.js (wersja 18+).
2. Skopiuj plik .env.example do .env i wstaw własny klucz API:
   - OPENWEATHER_API_KEY (OpenWeatherMap wymaga klucza autoryzującego).
3. Uruchom:
   - npm install
   - npm start

Skrypt wykona:

- GET do https://jsonplaceholder.typicode.com/todos/1
- POST do https://jsonplaceholder.typicode.com/posts
- GET do OpenWeatherMap (wymaga klucza API)
